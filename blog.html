Question-1: What are the different ways to select an element in the DOM?
There are many ways to select an element in the dom, such as document.getElementById(), getElementsByClassName() etc aswell as querySelector() or querySelectorAll() with which u can use css selectors to get an element or elements
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML means everything inside the element including html tags & text while innerText means all the text we are able to see on the screen, textContent means every text including comments on the HTML
Question-3: What is event delegation in the DOM?
Event Delegation is a method to efficiently use events and not have to repetitively create new and new eventlisteners and instead use one for all elements that need it.
Question-4: What is event bubbling in the DOM?
Event Bubbling is a method where the event gets bubbled back through every node in the DOM, For example if we trigger an event in a structure like html > body > div > button and had an event on the button it would bounce through these elements
Question-5: How do you create, add, and remove elements using JavaScript?
We add elements with document.createElement() and we add it to the HTML by using appendChild, we remove an element with remove()
